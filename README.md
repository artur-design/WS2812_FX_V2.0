# WS2812_FX_V2.0
Эффекты для адресной светодиодной ленты на ардуино.
Это форк проекта от AlexGyver, который находится по адресу https://github.com/AlexGyver/WS2812_FX 
# Изменения
Данный форк включает следующие изменения:
* __Оптимизация.__  Убраны лишние переменные, которые увеличивали объем используемой оперативной памяти почти в 3 раза. Оптимизированы режимы
* __Функционал.__  Добавлено ещё несколько режимов, среди которых мой собственный - градиент, а также режим, который я взял по адресу https://github.com/marmilicious/FastLED_examples/blob/master/sparkles.ino, убрано несколько режимов, которые посчитал вообще не привлекательным (не все), поскольку вначале создавал для себя. В последующих версиях (если будут) планирую вернуть убранные режимы в закомментированном виде, чтобы, если кому захочется, он мог их использовать. Также добавлены параметры для режимов, делающие их более красочными и интересными.
# Возможные улучшения бесконечны
Можно и дальше оптимизировать, делая их более стабильным в работе и более компактными, и добавлять различные режимы, делать их все более и более красивыми
